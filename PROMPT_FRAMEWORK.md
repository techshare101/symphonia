# Prompt Framework

### 1. Context
Symphonia creates setlists as stories, aligning music tempo with lyrical/emotional arcs.

### 2. User Journey
1. Upload music folder  
2. AI analyzes tempo + lyrics  
3. Choose arc template (Hero's Journey, Romance Arc, etc.)  
4. AI outputs setlist + .srt subtitles (FR/ENG/ES)  
5. Export to DJ software or stage visualizer  

### 3. Tech/API Spec
- Librosa/Essentia (audio features)  
- Whisper (lyrics)  
- GPT-5 (narrative arc + translations)  
- Supabase (storage)  

### 4. Design Direction
Cosmic gradients, spiral arcs, trilingual subtitle bars.
